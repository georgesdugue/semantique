Building a CSS Framework from Scratch

Published on January 15, 2025 by Jane Developer

Creating a CSS framework might seem daunting at first, but with the right approach, it can be a rewarding experience. In this post, I'll walk you through the key decisions and techniques that make a classless framework work.

Why Classless?

The main advantage of a classless approach is simplicity. Developers can focus on writing semantic HTML without worrying about which classes to apply. The framework handles the visual presentation automatically.

"The best CSS is the CSS you never had to write."

— A wise developer

Key Techniques

Here are the main techniques we used:

  1. CSS Custom Properties: Variables make theming trivial
  2. The :has() Selector: Enables parent-based styling
  3. Semantic HTML5: Use the right elements for the job

Example: Automatic Card Grid

Using :has(), we can detect when a section contains articles and automatically apply a grid layout:

section:has(> article) {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 1.5rem;
}
